Sample App: Signal R with CosmosDB Integration via Managed Service Identity

This is a simple app to show how to use Signal R and CosmosDB. You can either broadcast messages to all other users, or simulate an long running operation that requires notification on finish (e.g. placing an order). Orders are written to CosmosDB and then 5 seconds later, the submitter will be notified via Signal R that the order has been processed. See README for further information.
